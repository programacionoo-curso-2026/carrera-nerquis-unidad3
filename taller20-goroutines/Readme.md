# Taller 20 - Goroutines

## Objetivo

Practicar el uso de goroutines en Go para ejecutar tareas concurrentes.

## Estructura

- README.md
- src/main.go

## Ejecución

```bash
go run src/main.go
```

## Descripción

El programa crea diez goroutines. Cada una espera un tiempo aleatorio antes de finalizar e imprime su identificador junto con el tiempo de espera.
